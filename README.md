# RSK Auto Detailing

Premium auto detailing website — Hamilton & GTA.

## Structure
```
rsk-detailing-site/
├── index.html        # markup + content
├── css/
│   └── styles.css    # all styles
├── js/
│   └── main.js       # interactivity (cursor, scroll animations, etc.)
└── assets/           # images, video, 3D models (add here)
```

## Run locally
```bash
python3 -m http.server 5500
```
Then open http://localhost:5500

Or use the VS Code Live Server extension.
